猛虎战区

训练场北部最重要的岛屿，列岛战区的同胞拥有不屈的品质，是最让其他战区头疼的对手。

列岛战区

占据了全息训练场北方的大部分土地，广阔的平原赋予猛虎同胞们坦荡的胸怀和无人能敌的勇气。

绿叶战区

训练场东部的明珠，山势平缓宜居。这里的人们坚信公平公正的比赛环境才能让人发挥最大实力。

神雷战区

位于全息训练场的北部半岛。活跃在这里的人们信仰正义与荣耀，誓用高超的战斗技巧扫尽一切黑暗和阴霾。

圣象战区

训练场南海中的岛屿，遍布河滩与树林。圣象战区的战士们勤奋团结，是最暖心的队友和最危险的对手。

影狼战区

拥有平缓的丘陵和清澈的河流。影狼同胞擅长以出人意料的战术给予敌人致命打击。

远峰战区

最具特色的战区，分布着高山和峡谷。远峰同胞们拥有永不言弃的意志力，这是在训练场中获胜的根本。
